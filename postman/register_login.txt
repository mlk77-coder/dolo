 How to Test with Postman:
I've created a comprehensive guide: POSTMAN_API_TESTING_GUIDE.md

Quick Start:

Start your server:

php artisan serve
Register a new user:

URL: POST http://localhost:8000/api/register
Headers:
Content-Type: application/json
Accept: application/json
Body (JSON):
{
    "name": "Ahmad",
    "surname": "Hassan",
    "phone": "9123456789",
    "email": "ahmad@example.com",
    "date_of_birth": "1990-01-15",
    "gender": "male",
    "password": "password123",
    "password_confirmation": "password123"
}
Login with email:

URL: POST http://localhost:8000/api/login
Body (JSON):
{
    "credential": "ahmad@example.com",
    "password": "password123"
}
Login with phone:

URL: POST http://localhost:8000/api/login
Body (JSON):
{
    "credential": "9123456789",
    "password": "password123"
}
Get user profile (authenticated):

URL: GET http://localhost:8000/api/user
Headers:
Authorization: Bearer YOUR_TOKEN_HERE
Accept: application/json
